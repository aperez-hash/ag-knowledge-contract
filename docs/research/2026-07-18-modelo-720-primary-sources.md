## 2.5 Nota interpretativa para el objeto

1. La base regulatoria del Modelo 720 debe leerse hoy en conjunto:
   - `Ley 58/2003` -> fundamento legal;
   - `Real Decreto 1065/2007` -> detalle material de las tres obligaciones;
   - `Orden HAP/72/2013`, en su redaccion vigente -> aprobacion del modelo y procedimiento formal;
   - `Orden HFP/1180/2023` -> actualizacion posterior relevante del modelo.
2. La sentencia del TJUE no elimina la obligacion informativa del Modelo 720; condiciona el tratamiento del regimen sancionador y obliga a no describir como vigente un marco ya corregido normativamente.

## 4. Criterios de uso de estas fuentes

1. El objeto debe limitarse a la obligacion de presentar el Modelo 720.
2. El modelo 721, CRS, FATCA, Modelo 714 o residencia fiscal como materia
   autonoma no deben desarrollarse aqui, aunque puedan citarse como relaciones
   planificadas.
3. La FAQ y el Client Response deben derivarse del mismo objeto, no de texto
   paralelo.
4. El bloque `LegalBasis` debe explicar efecto practico y alcance, no copiar
   literalmente la norma.
5. El impacto del TJUE debe aparecer solo como criterio operativo actual:
   especialmente para no sostener sanciones o efectos ya invalidados o
   corregidos.
